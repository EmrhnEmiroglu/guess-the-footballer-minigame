# 📋 Proje Özeti

## Proje Adı
**Oyuncu Tahmin Oyunu** — Günlük Futbolcu Wordle Eklentisi

## Sürüm
**2.0.0** (Stable)

## Lisans
GPLv2 or later

---

## 🎯 Amaç

Futbol severlere **Wordle tarzında günlük tahmin oyunu** sunmak:
- Her gün aynı 5 futbolcu (2 kolay + 2 orta + 1 zor)
- Blurlu görsel → giderek netleşir
- Wordle mekanizması
- Seri takibi

---

## 🔧 Teknoloji Stack

**Backend**: PHP 7.4+ | WordPress 5.8+ | REST API
**Frontend**: Vanilla JS | CSS3 | Responsive
**Database**: Custom Post Type + Meta fields + Transient cache
