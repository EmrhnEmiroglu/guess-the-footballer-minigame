# 🤝 Katkı Rehberi

Oyuncu Tahmin Oyunu projesine katkıda bulunmak istediğiniz için teşekkürler!

## 🐛 Hata Bildirme

GitHub Issues'de oluştur:
1. **Başlık**: Kısa & açıklayıcı
2. **Açıklama**: Detaylı hata
3. **Adımlar**: Tekrar üretme yöntemi
4. **Ortam**: WordPress, PHP, tarayıcı versiyonları

## 💡 Özellik İsteği

1. Neden ihtiyaç olduğunu açıkla
2. Olası çözüm örneği ver

## 🔧 Pull Request Süreci

### 1. Fork & Branch
```bash
git clone https://github.com/YOUR_USERNAME/oyuncu-tahmin-oyunu.git
cd oyuncu-tahmin-oyunu
git checkout -b feature/yeni-ozellik
```

### 2. Kod Yaz
- Yorum ekle
- Test et
- Standartları takip et

### 3. Commit
```bash
git commit -m "feature: yeni özellik açıklaması"
```

Formatlar:
- `feature:` — Yeni özellik
- `fix:` — Hata düzeltme
- `docs:` — Dokümantasyon
- `refactor:` — Kod yeniden yazma

### 4. Push & PR
```bash
git push origin feature/yeni-ozellik
```

GitHub'da "New Pull Request" tıkla

## 📋 Kod Standartları

### PHP
- PSR-12 takip et
- 4 boşluk indentan
- `oto_function_name()` formatı

### JavaScript
- `const` / `let` kullan
- `camelCase` değişken adları
- `'use strict';` başında

### CSS
- BEM metodolojisi
- Custom properties
- Mobile-first
